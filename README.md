# Money pattern
    Money java implementation from scratch. Manually compiling and building everything. No build tools. Good old javac. 

    Important java packages:
        java.lang (core)
        java.util
        java.io(stream)/nio(buffer)
        java.sql
        java.math
        javax.persistence
